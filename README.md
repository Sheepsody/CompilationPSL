# 言語

## Syntaxe d'un programme

```shell
/*
Commentaires
*/

# Variables
let a = 3;
let b = 3;
a = b = 1;

# Function
fn (a, b) {
    let a = 5;
    5 # Retour implicite sans ;
}

# If-Else condition
if cond {
    ... ;
    ... ;
} else {
    ... ;
    ... ;
}

# While loop
while cond {
    ... ;
}

5 # Retour implicite sans ;
```
