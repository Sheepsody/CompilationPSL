# 言語

## Syntaxe d'un programme

```shell
# Variables
let a = 3;
let b = 3;
a = b = 1;

# Function
fn (a, b) {
    let a = 5;
    5 # Retour implicite sans ;
}

# If-Else condition
if (cond) {
    ... ;
    ... ;
} else {
    ... ;
    ... ;
}

5 # Retour implicite sans ;
```
